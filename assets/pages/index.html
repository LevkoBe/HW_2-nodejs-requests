Where ducks, hens, and geese live peacefully in the digital world!

API Endpoints:

  • / - Get general information
  • /info - Get API endpoints information
  • /story - Explore an intriguing story
  • /image - Take a look at an image
  • /file/:filename - Get the content of a text file
  • /objects/:character/:type - Get a specific object by character and animal types
  • /objects/:character - Get objects by character type
  • /objects - Get all objects
